# Google Maps Hospital Scraper

## Project Overview

This project is a Google Maps scraper built using Python and Selenium.

The scraper searches for hospitals in Chennai on Google Maps, automatically scrolls through the results, extracts business information, and exports the data into a CSV file.

## Features

* Search businesses on Google Maps
* Automatic scrolling of search results
* Extract business information
* Export data to CSV
* Handle dynamically loaded content

## Extracted Fields

* Business Name
* Rating
* Reviews Count
* Phone Number
* Website URL
* Google Maps URL

## Technologies Used

* Python
* Selenium
* Pandas
* ChromeDriver

## Workflow

1. Open Google Maps
2. Search for "Hospitals in Chennai"
3. Automatically scroll through results
4. Collect business cards
5. Extract required information
6. Store data in a Pandas DataFrame
7. Export data to CSV

## Output

The scraper generates a CSV file containing hospital information collected from Google Maps.
